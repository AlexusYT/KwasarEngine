What is this page about?

Developing a game engine, like any other software development process, is a very laborious task that requires time. Currently, the project is created, maintained, and improved by one person's efforts, which obviously isn't enough for rapid development. If you have the desire and ability to help, I would be happy to receive your contribution.

I want to help but don't know how

There are actually many options, it all depends on the person, their skills, and abilities:

  • If you are a C++ programmer, you can check the issues section of the repository for bugs or development tasks. Also, the file CONTRIBUTING.md will be useful for you.
  • If you are a tester, your task is to test the editor's interface and API of the engine, as well as their results (compiled games). Bug reports should be created here.
  • If you are a game developer, we ask you to leave feedback and suggestions for improving the engine, its parts, and the editor here.
  • And if you've read this far and nothing seems suitable for you, that's okay. There is also a task for you: share this site with those who might be interested.

Thank you very much for your help to the project!
