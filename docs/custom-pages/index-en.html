About the Project

Kwasar Engine (formerly MegaEngineRing) is a game engine developed in C++.

Status

The engine is in early development stage. If you would like to help the project, you can find all information about support options (not about money) on this page.

Documentation

The documentation is divided into 3 parts:

  • Engine User Manual is intended for regular users. It contains information about how to use the editor, what features and limitations the application has.
  • Engine API and Code Documentation is needed for developers and advanced engine users.
  • Editor Code Documentation is mainly used by editor developers.

In the early stage, documentation and manual may lag behind the actual state of affairs, but the Russian version will always be more up-to-date than other versions. If the documentation does not match reality, please create an Issue in the engine repository describing the discrepancy and mark it with "documentation" tag - this way you'll help the project development.

System Requirements

  • Operating System: Windows 10+ or Ubuntu 22.04+
  • RAM: 4GB+
  • Graphics card with OpenGL 4.5 support, i.e., any modern GPU
