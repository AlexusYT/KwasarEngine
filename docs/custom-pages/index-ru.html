О проекте

MegaEngineRing — это игровой движок, разработанный на языке C++.

Статус

Движок находится в ранней стадии разработки. Если у вас есть желание помочь проекту, на этой странице собрана вся информация о вариантах поддержки (речь не о деньгах).

Документация

Документация разделена на 3 части:

  • Руководство по использованию движка предназначено для обычных пользователей. Здесь содержится информация о том, как пользоваться редактором, какие есть возможности и ограничения приложения.
  • Документация к коду API движка нужна для разработчиков и продвинутых пользователей движка.
  • Документация к коду редактора используется в основном разработчиками редактора.

На раннем этапе, документация и руководство могут отставать от реального положения дел, но русскоязычная версия всегда будет актуальнее других версий. В случае если документация не соответствует действительности, создавайте Issue в репозитории движка с описанием несоответствия и пометкой "documentation", так вы поможете развитию проекта.

Системные требования

  • Операционная система: Windows 10+ или Ubuntu 22.04+
  • Оперативная память: 4ГБ+
  • Видеокарта с поддержкой OpenGL 4.5, то есть любая современная
